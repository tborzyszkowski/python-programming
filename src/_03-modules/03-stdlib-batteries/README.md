# 03 - Standardowa biblioteka (batteries included)

## Cel

Pokazac, ze Python posiada bogaty zestaw gotowych narzedzi: praca z plikami, strukturami danych, JSON, iteracjami i datami.

## Co oznacza batteries included?

W wielu jezykach trzeba od razu instalowac zewnetrzne biblioteki do podstawowych zadan. W Pythonie duza czesc tych potrzeb pokrywa standard library.

Przyklady z `examples/stdlib_showcase.py` uzywaja:
- `pathlib` - operacje na sciezkach,
- `collections.Counter` - zliczanie,
- `itertools` - laczenie iteratorow,
- `json` - serializacja danych.
